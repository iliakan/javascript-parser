# Якорь для ссылок #hash

Якоря позволяют легко ссылаться на любой заголовок статьи.
 
Заголовок с якорем выглядит так:
```
## Header [#cool-stuff]
```

Ссылка на него:
```
Смотрите [сюда](#cool-stuff)
```

Якоря могут быть только внутри статей.
